# 家計・資金管理アプリ（計画 vs 実績）

マネーフォワード ME のデータを取り込み、**月の計画（利用上限）と実績を色分けで見える化**する Web アプリです。
「ここまでは使ってよい」を日割りで把握し、ムダを省くことを目的にしています。

## コンセプト

```
月初：計画を立てる   → 現金・カードの利用上限、入金目標を決める
日々：実績を取り込む → マネーフォワード ME の CSV をドラッグするだけ
いつでも：見える化   → 計画 vs 実績 vs 着地予測を 🟢🟡🔴 で表示
```

## 主な機能

| タブ | 内容 |
|------|------|
| 📊 ダッシュボード | 残り日数 / 今日まで使ってよい額（日割り）/ 計画 vs 実績 vs 着地予測 / カテゴリ別の色分け |
| 🎯 月の計画 | 現金上限・カード上限・入金目標、カテゴリ別上限の設定 |
| 📥 実績取込 | MF ME の CSV を取込（Shift-JIS/UTF-8 自動判定・重複登録防止） |
| 🧾 明細 | 当月の明細一覧、現金の手入力、削除 |

## 使い方

1. `index.html` をブラウザで開く（スマホでも可）
2. 初回はメールアドレスで **新規登録**（確認メールが届く場合はリンクを開く）→ ログイン
3. 「🎯 月の計画」で現金・カードの上限を入力して保存
4. マネーフォワード ME →「家計簿」→「入出力」→ CSV ダウンロード
5. 「📥 実績取込」にその CSV をドラッグ＆ドロップ
6. 「📊 ダッシュボード」で計画との差を確認

## データ構成（Supabase / PostgreSQL）

すべて `expense_` 接頭辞で、既存データとは完全分離。

| テーブル | 役割 |
|---------|------|
| `expense_categories` | カテゴリマスタ |
| `expense_monthly_plans` | 月次計画（現金/カード上限・入金目標） |
| `expense_category_budgets` | カテゴリ別の上限 |
| `expense_transactions` | 実績明細（MF/CSV/手入力） |
| `expense_import_log` | 取込履歴 |

- RLS 有効。ログイン済みユーザーのみアクセス可（匿名アクセス不可）。
- 公開キー（`sb_publishable_...`）はブラウザ用の公開鍵で、コードへの埋め込みは想定どおり。

## ロードマップ

- **Phase 1（現在）**: 自分のコスト削減 — 計画 vs 実績の見える化
- **Phase 2（将来）**: 中小企業向けの資金繰り管理 — 複数人・承認フロー・残高予測
